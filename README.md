# k8s-homelab

This repository is a collection of Kubernetes configurations files for my homelab.  The homelab is hosted locally using minikube, and is used for testing and learning purposes. The configurations are organized into different folders based on the type of application or service they deploy.

## Setup
To set up the homelab, you will need to have minikube installed on your local machine. You can follow the instructions on the minikube website to install it: https://minikube.sigs.k8s.io/docs/start/?arch=%2Flinux%2Fx86-64%2Fstable%2Fbinary+download

Docker is also required to run minikube. You can install Docker by following the instructions on the Docker website: https://docs.docker.com/get-docker/

To start the minikube cluster, run the following command in your terminal:
```
minikube start
```
